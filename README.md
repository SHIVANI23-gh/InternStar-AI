# 🌟 IntenStar AI – AI-Powered Internship & Certification Platform

Live Link: https://asset-manager--shivani1023.replit.app

IntenStar AI is a full-stack web application that helps students discover internships and professional certification opportunities from multiple trusted platforms in one place. The platform uses Artificial Intelligence to provide personalized recommendations, making it easier for students to find opportunities that match their skills, interests, and career goals.

---

## 🚀 Features

- 🔍 Search internships and certifications
- 🤖 AI-powered personalized recommendations
- 📂 Filter opportunities by category, location, duration, and mode
- ❤️ Save favorite opportunities
- 👤 User authentication and profile management
- 📊 Interactive dashboard
- 📱 Responsive design for desktop and mobile
- ⚡ Fast and modern user interface

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/Asset-Manager.git
```

### Navigate to the Project

```bash
cd Asset-Manager
```

### Install Dependencies

Frontend

```bash
cd client
npm install
```

Backend

```bash
cd ../server
npm install
```

---

## ▶️ Run the Application

Start Backend

```bash
npm start
```

Start Frontend

```bash
npm run dev
```

The application will be available at

```
http://localhost:5173
```
